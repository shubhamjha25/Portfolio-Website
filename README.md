### RUN THE SITE LOCALLY


```bash
git clone https://github.com/shubhamjha25/Portfolio-Website.git
```

```bash
cd Portfolio-Website
```
```bash
npm i && npm start
```

After running the development server the site should be running on https://localhost:3000
